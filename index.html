Neuroimaging Agentic Analysis Framework

LLM-powered agent for automated EEG/MEG preprocessing with visual inspection. Let AI handle the pipeline while you focus on the science.

Get Started View on GitHub

Capabilities

Intelligent preprocessing,
zero boilerplate

NAAF uses Claude to autonomously reason about your neuroimaging data, make preprocessing decisions, and produce publication-ready results.

🧠

Agentic Analysis

An LLM agent reasons about your data, selects appropriate preprocessing steps, and adapts the pipeline to your specific recordings.

👁

Visual Inspection

Automated generation of quality control plots so you can verify every decision the agent makes — trust, but verify.

🔒

Non-destructive

Raw data is never modified. All outputs and intermediate results are cleanly organized in your project folder.

⚡

Multi-format Support

Works with BrainVision, EDF, BDF, FIFF, and EEGLAB formats out of the box — no conversion needed.

🤖

Mock Mode

Test the full pipeline without an API key using simulated LLM responses — perfect for evaluation and development.

💬

Interactive Mode

Guide the agent step by step with interactive prompts, maintaining full control over preprocessing decisions.

Quick Start

Up and running in under a minute

Requires Python 3.10+ and git. All dependencies are installed automatically.

.vhdr .edf .bdf .fif .set
terminal
# create a virtual environment
mkdir MyProject && cd MyProject
python -m venv .venv
source .venv/bin/activate

# install NAAF
pip install git+https://github.com/kmahjoory/naaf.git

# set your API key
export ANTHROPIC_API_KEY='your-key'

# run
naaf --project ./output --data /path/to/eeg/

Workflow

How it works

NAAF orchestrates the entire preprocessing pipeline through an intelligent agent loop.

Point to your data

Provide a path to your raw EEG/MEG recordings and a project directory for outputs. NAAF auto-detects the format.

Agent reasons & acts

The LLM agent inspects your data, selects preprocessing steps (filtering, artifact rejection, ICA, etc.), and executes them.

Review & iterate

Quality control plots are generated at each stage. Use interactive mode to guide decisions or let the agent run autonomously.
