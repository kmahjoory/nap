NAP

Neuroimaging Agentic Analysis Platform

Prototype v0.1.0

An LLM-powered agent that autonomously preprocesses EEG/MEG data through visual inspection and intelligent artifact rejection.

Visual Inspection

Generates diagnostic plots and sends them to a multimodal LLM for analysis. The agent sees what a researcher sees.

Autonomous Processing

Iterative artifact rejection loop: the agent proposes actions, executes them, evaluates results, and repeats until clean.

Read-Only Safety

Raw data is never modified. All work happens on in-memory copies. Outputs go to a separate project folder.

How It Works

1
Organize — Point NAP at a data folder. It scans, classifies files, and creates a structured workspace with symlinks.
2
Inspect — Generates PSD and trace plots. The LLM examines them to identify bad channels, segments, and line noise.
3
Plan — Based on visual findings, the agent proposes a preprocessing plan and saves it for review.
4
Clean — Autonomous artifact rejection: ICA, channel removal, segment annotation, notch filtering — iterating until done.
5
Report — Every decision is logged. A full report is saved for researcher review.

Get Started

# Install
pip install git+https://github.com/kmahjoory/nap.git

# Run
export ANTHROPIC_API_KEY='your-key'
nap --project ./output --data /path/to/eeg/

# Without API key (mock mode)
nap --mock --project ./output --data /path/to/eeg/
